# Quickstart (1 minute)

## 1) Clone and install
```bash
git clone https://github.com/alaidas/parking.git
cd parking
npm install
```

## 2) Start app
```bash
npm start
```
Open: http://localhost:3000

## 3) First run (bootstrap)
- In browser, set admin password in bootstrap modal.
- Login with:
  - user: `admin`
  - password: the one you just set

## 4) Main usage
- Pick floor + date.
- Select spot.
- Book / release based on your role.
- Open **Admin Panel** for users/floors/spaces management.

## Notes
- DB file: `data/parking.sqlite3`
- DB key: `secrets/db-access.key`
- Backup both files together.
